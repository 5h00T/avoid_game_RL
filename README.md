# avoid_game_RL

# Usage
このレポジトリに含まれている学習済みモデルのプレイを見たい場合
```
python v0playT1.py [--save] [--times <number>] AI
```

手動でプレイしたい場合
```
python v0playT1.py [--save] [--times <number>] Human
```
